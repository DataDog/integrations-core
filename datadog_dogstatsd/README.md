# Datadog DogStatsD

## Overview

This integration surfaces the [client-side telemetry][2] emitted by Datadog's official [DogStatsD][1] client libraries. These metrics are generated by the clients themselves and provide visibility into the entire metric submission lifecycle. They track how many metrics, events, and service checks the application sends to the client, how many payloads and bytes are successfully delivered to the Agent, and how many are dropped because of a full sender queue or a writer error. When client-side aggregation is enabled, aggregation context counts are also reported.
