# APM Config Audit

## Overview

APM Config Audit provides an out-of-the-box dashboard to help you spot hostname configuration issues in your APM setup, such as services emitting pod-like or empty hostnames.

## Setup

Install the integration to get the **APM Config Audit** dashboard. The dashboard reads from existing APM data, so no additional configuration is required. Guidance for resolving any issues is included directly in the dashboard.
